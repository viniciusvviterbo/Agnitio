# Agnitio

Software para manipulação de imagens.

### Instalando as Dependências

1. (Opcional) Crie um ambiente virtual:
```shell
virtualenv .venv
```

2. Clone esse repositório e execute:
```shell
pip3 install -r requirements.txt
```

### Uso

```
python3 agnitio.py
```
